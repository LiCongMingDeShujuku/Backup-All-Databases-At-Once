## Logic
```SQL
use master;
set nocount online

exec master..sp_msforeachdb
'if (''?'') not in (''tempdb'')
	begin
		backup database [?] to disk = ''D:\MyBackupFolder\?.bak'' with format, checksum;
	end'

```

This could easily be enhanced further by adding time stamps and some additional logic that checks certain proceses, or verifies the backups using checksum.
